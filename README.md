# Order-Delivery-System-SQL-Case-Study

##  Objective
Analyze customer orders and delivery performance to extract business insights using SQL.

##  Tools Used
- SQL

##  Key Business Questions
- Which city generates the highest revenue?
- Who are the top customers?
- Which delivery partners perform best?
- What is the average delivery time?

##  SQL Concepts Used
- INNER JOIN, LEFT JOIN
- Subqueries
- Window Functions
- Aggregations

##  Key Insights
- Metro cities contribute highest revenue
- High-value customers identified for loyalty programs
- Delivery delays vary significantly by partner
